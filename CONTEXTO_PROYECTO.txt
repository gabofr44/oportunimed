1. DESCRIPCIÓN GENERAL
========================================
Oportunimed (antes "Global Pathways") es una plataforma bilingüe para que estudiantes
encuentren becas, pasantías, fellowships y oportunidades de investigación internacional.
Todo es gratuito (GitHub Student, Vercel free, Supabase free, Upstash free).

- Repo: https://github.com/gabofr44/oportunimed.git
- Frontend: https://oportunimed.vercel.app
- Supabase: https://tloxuzwnjjlwlkdkfsvr.supabase.co
- Admin password: oportunimed2026

2. STACK TECNOLÓGICO
========================================
- Next.js 16 (App Router)
- Supabase (Auth + PostgreSQL)
- Tailwind CSS v4
- Framer Motion (animaciones)
- Lucide React (iconos)
- Upstash Redis (rate limiting)

5. ESQUEMA DE BASE DE DATOS
========================================
TABLA profiles:
  - id (UUID, FK auth.users)
  - full_name, avatar_url, bio, country, university
  - role (student/admin)

TABLA categories:
  - id (UUID)
  - name, slug, description, icon
  - Categorías existentes en full_setup.sql:
    * becas (scholarships)
    * pasantias (internships)
    * becas-internacionales (international scholarships)
    * investigacion (research)
    * programas-profesionales (professional programs)
    * oportunidades-STEM (STEM opportunities)
    * recursos-idiomas (language resources)

TABLA opportunities:
  - id (UUID)
  - title, institution, location
  - type (ENUM: scholarship, research, internship, course)  ← USA ESTE, NO category_id
  - funding (BOOLEAN)                                      ← USA ESTE, NO is_funded
  - description (TEXT)
  - link (TEXT)
  - deadline (TIMESTAMPTZ NOT NULL)                        ← REQUERIDO
  - tags (TEXT[])
  - is_featured (BOOLEAN)
  - created_by (UUID, FK profiles)
  - educational_level (TEXT)   ← Agregado por migration 005
  - educational_field (TEXT)   ← Agregado por migration 005
  - country_code (TEXT)

TABLA applications:
  - id (UUID)
  - user_id, opportunity_id
  - status (pending/accepted/rejected/withdrawn)
  - notes, applied_at
